# MVP 范围说明（InterviewMate RAG）

## 项目目标

构建一个可在本地快速演示的 **RAG 辅助技术面试练习** 小工具，用于简历展示：选题与难度、生成题目、对照参考材料、提交答案后获得结构化评分与反馈。

## MVP 功能

- 固定主题与难度枚举，从本地 JSON 种子中抽取题目。
- 启动时使用 OpenAI Embedding 对种子条目建 **FAISS** 向量索引，用于**评卷**时的检索证据（出题阶段仅按主题与难度随机抽题，不做向量检索）。
- 评卷调用大模型，按 rubric 输出 **JSON**（分数、亮点、缺失、改进回答）。
- 单页前端：选题、生成、展示要点与参考、填写答案、评估结果展示。

## 非目标（刻意不做）

- 用户体系、鉴权、权限角色、管理后台。
- 支付、职位搜索、简历解析、大规模生产运维。
- 外部数据库（首版仅本地 JSON 文件）。

## 演示局限

- 需要可用的 **OpenAI API Key**（Embedding + Chat），无离线降级。
- 题库规模小，检索与评分仅供演示，不代表真实面试评分标准。
- CORS 仅放行本地前端常用端口，非生产安全配置。
